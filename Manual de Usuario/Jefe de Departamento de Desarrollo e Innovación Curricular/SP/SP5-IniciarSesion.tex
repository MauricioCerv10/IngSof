    \section{Ingresar al sistema}
        Para poder iniciar sesión en el sistema, debe ingresar la url: \url{http://} y podrá ver la siguiente pantalla:
        
        % Imagen de Iniciar sesión
        \begin{figure}[!hbtp]
            \centering
            \hypertarget{iniciarS}{\includegraphics[width=0.7\linewidth]{images/SP5/IniciarSesion}}
            \caption{Pantalla para iniciar sesión}
            %\label{consultarrh}
        \end{figure}
        
        Para poder ingresar sesión debe tener su matricula y contraseña que su Jefe de la División de Innovación Académica le dio, dichos datos se ingresarán como en el ejemplo que se muestra abajo.
        
        % Imagen de Iniciar sesión con llenado 
        \begin{figure}[!hbtp]
            \centering
            \hypertarget{iniciarL}{\includegraphics[width=0.7\linewidth]{images/SP5/ejemploIniciar}}
            \caption{Pantalla iniciar sesión de ejemplo}
            %\label{consultarrh}
        \end{figure}
        
        Después de ingrésalos debe seleccionar el botón:
        
        % Botón Ingresar
        \begin{figure}[!hbtp]
            \centering
            \hypertarget{BotonIng}{\includegraphics[width=0.7\linewidth]{images/SP5/BotonIngresar}}
